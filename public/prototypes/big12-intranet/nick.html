About

College sports has been run on the same spreadsheet stack for forty years. I'm replacing it.

I'm Director of Competition at the Big 12 Conference and founder of HELiiX AI Solutions — a studio building operational software for collegiate athletics. By day I run championships across Wrestling, Tennis, and Men's Basketball, and oversee scheduling for ten Olympic sports. By night I write the code that makes the next day easier.

"Almost-right is the most expensive software an operations team can buy."

The tools the conferences buy are almost-right. They never quite fit the way the work actually moves. After a decade running ops at Ball State, Incarnate Word, the Spurs, and now the Big 12, I started building the missing layer myself. FlexTime — the CP-SAT engine that now produces Big 12-approved conference schedules — was the first piece. Around it grew governance pipelines, awards processing, tournament seeding, tiebreakers, and the kind of validation an ops director can actually trust at 11pm the night before bracket release.

I work the way an operator would build software if they had time: domain-deep, explainable, deployed in increments, written so the next person can read it. HELiiX takes on a small number of projects each year. Mostly collegiate athletics. Always operational.

I live in Texas with my wife Sam and our daughter Gianna.
